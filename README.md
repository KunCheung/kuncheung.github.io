# kuncheung-homepage

A personal homepage built with Vite and React. The site is content-first and includes an intro, updates, publications, writing, and contact links.

## Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Content

- Homepage content is maintained in the data arrays in `src/App.jsx`, such as `updates`, `posts`, `publications`, and `links`.


## Deployment Notes

- `dist/` is generated by `npm run build`; do not edit it manually.
- If deploying to a GitHub Pages project URL such as `https://<username>.github.io/kuncheung/`, set the matching `base` value in `vite.config.js`.
- If deploying to a root user site such as `https://<username>.github.io/`, the default `base` is usually enough.
